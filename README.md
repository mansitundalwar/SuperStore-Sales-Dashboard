📊 SuperStore Sales Dashboard & Forecast

Project Overview
This project is an interactive Power BI dashboard built on SuperStore dataset.
It provides insights into sales, profit, and includes a 15-day sales forecast.

 Features
- Sales and Profit Analysis
- Region-wise and Category-wise insights
- Sales by Payment Mode & Ship Mode
- Monthly YOY Trends
- 15-Day Sales Forecast using time series

 Tools Used
- Power BI
- Excel
